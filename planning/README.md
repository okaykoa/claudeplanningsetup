# Planning

Project planning docs for use as context with AI agents. Fill them out in this order: principles → design → requirements → roadmap. In any agent configuration files (e.g. `CLAUDE.md`), point to this directory as a context source.

| Doc | Purpose |
|---|---|
| `principles.md` | Guiding values and philosophy behind the project |
| `design.md` | The functionality, data, and model that define the product |
| `requirements.md` | Technical requirements and implementation decisions |
| `roadmap.md` | Milestones, steps, and definitions of done |
